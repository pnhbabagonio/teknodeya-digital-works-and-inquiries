# Teknodeya Service Inquiry Management System

A production-ready web application for managing service inquiries, built with Next.js 16, Supabase, and Tailwind CSS.

## Features

- 🎨 Modern, premium dark UI with Teknodeya branding
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🔐 Full Supabase authentication integration
- 📝 Comprehensive inquiry form with file uploads
- 👨‍💼 Admin dashboard with complete CRUD operations
- 📊 Real-time statistics and analytics
- 🔍 Filterable, sortable, paginated inquiries table
- 📎 File attachments with preview functionality
- ⚡ Smooth animations with Framer Motion
- ✅ Form validation with Zod
- 🎯 SEO optimized

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Forms**: React Hook Form + Zod
- **Animations**: Framer Motion
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **File Storage**: Supabase Storage
- **Icons**: Lucide React

## Prerequisites

- Node.js 18.17 or later
- npm or yarn or pnpm
- Supabase account
